About DMwithAI Demo Store

Purpose

This is a demo website built specifically to test the DMwithAI analytics modules:

  • Cohort Retention Check - Track user retention over time by signup date, channel, and campaign
  • Audience Segmentation - Identify high-performing audience segments based on behavior and demographics

How It Works

Every page includes a tracking snippet that automatically sends events to the DMwithAI collector:

  • Pageviews - Tracked on every page load
  • Engagement - Triggered after 30 seconds or 75% scroll
  • Bounces - Detected on quick exit without engagement
  • Conversions - Triggered by button clicks (newsletter signup, add to cart, purchase)

Testing Scenarios

You can test various scenarios:

  • Navigate between pages to generate multiple pageviews
  • Use different UTM parameters to simulate different traffic sources
  • Click conversion buttons to generate conversion events
  • Return to the site later to test returning visitor detection
  • Generate enough sessions (100+) to enable audience segmentation

Traffic Sources

Test different acquisition channels by using these UTM parameters:

  • ?utm_source=google&utm_medium=organic - Organic search
  • ?utm_source=google&utm_medium=cpc&utm_campaign=summer_sale - Paid search
  • ?utm_source=facebook&utm_medium=social&utm_campaign=brand_awareness - Social media
  • No parameters - Direct traffic
